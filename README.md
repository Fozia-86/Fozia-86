## 🏆 Featured Capstone Project

### 📋 [Smart Invoice Auditor & Autonomous Settlement Agent](https://github.com/[Fozia-86]/invoice-auditor-agent)
A production-grade, cloud-deployed Multi-Agent AI system built during the **Kaggle 5-Day AI Agents Intensive (July 2026)**.
- **Extractor Agent:** Processes raw multi-page invoice assets (images/PDFs) using Gemini Multimodal capabilities to output structured JSON data.
- **Compliance Auditor Agent:** Uses the **Model Context Protocol (MCP)** to enforce spending limits, verifies vendor lifecycles via UCP, and signs automated payment mandates via **Agent Payments Protocol (AP2)**.
- **Data & Telemetry:** Streams operational metadata and live execution analytics to Cloud Firestore and Google BigQuery.
